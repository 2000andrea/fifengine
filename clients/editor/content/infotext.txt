This is the standalone FIFE editor, release 2008.0

Keybindings:
--------------
- P = Make screenshot
- LEFT = Move camera left
- RIGHT = Move camera right
- UP = Move camera up
- DOWN = Move camera down
- F10 = Toggle console on / off
- ESC = Quit techdemo

- M = Make a new instance on the map with
      current object selection at current
      mouse selection.
- X = Remove instances at current mouse
      selection. 
- T = shows / hides grid
- B = shows / hides blocking info
